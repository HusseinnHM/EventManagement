# EventManagement

Web application RESTful API builds using .NET Core 8 supports:
- JWT Authentication.
- Multi Languages Database.
- Handle Race Conditions.

*Developed using packages I already have build them:*

- [Neptunee.OperationResponse](https://github.com/HusseinnHM/Neptunee.OperationResponse) 
- [Neptunee.EntityFrameworkCore.MultiLanguage](https://github.com/HusseinnHM/Neptunee.EntityFrameworkCore.MultiLanguage) 
- [Neptunee.BaseCleanArchitecture](https://github.com/HusseinnHM/Neptunee.BaseCleanArchitecture) 
- [Neptunee.Swagger](https://github.com/HusseinnHM/Neptunee.Swagger)
